##############
 Getting help
##############

If you run into a problem using the IDF Component Manager, include the information below when asking for support.

*****************
 What to include
*****************

- Your ESP-IDF version
- Your Component Manager version (``compote version``)
- The command you ran (for example, ``idf.py add-dependency ...``)
- The full error output (copy/paste)

**************
 Where to ask
**************

- GitHub issues: `idf-component-manager <https://github.com/espressif/idf-component-manager/issues>`_
- Email: components@espressif.com
