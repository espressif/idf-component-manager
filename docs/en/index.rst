################################################################
 IDF Component Manager and ESP Component Registry Documentation
################################################################

This documentation covers the IDF Component Manager and the ESP Component Registry. The Component Manager integrates with ESP-IDF to download and manage components from the Registry, and it also provides the `compote` CLI for advanced workflows. The ESP Component Registry (`components.espressif.com <https://components.espressif.com>`_) is the central repository for ESP-IDF-compatible components.
